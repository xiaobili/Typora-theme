# Claude Anthropic - Typora Theme

一个温暖、富有编辑感的 Typora 主题系列，灵感来自 Claude (Anthropic) 的设计系统。

**🎨 现在提供两个版本：浅色（羊皮纸）和暗色（深暖色调）**

## 📦 可用主题

### ☀️ Claude Anthropic（浅色版）
- **文件**: `claude-anthropic.css`
- **特点**: 温暖的羊皮纸色调，适合日间使用
- **背景**: `#f5f4ed` (Parchment)
- **适用**: 明亮环境、传统阅读体验

### 🌙 Claude Anthropic Dark（暗色版）
- **文件**: `claude-anthropic-dark.css`
- **特点**: 温暖的深色调，适合夜间使用
- **背景**: `#141413` (Deep Dark)
- **适用**: 昏暗环境、减少眼部疲劳

> 💡 **提示**: 您可以同时安装两个版本，根据需要切换！查看 [主题对比文档](THEME_COMPARISON.md) 了解详细差异。

## ✨ 特色功能

### 🎨 视觉设计
- **独家暖色调中性色**：所有灰色都带有黄棕底色，完全避免冷蓝灰色
- **Terracotta 品牌色**：使用温暖的赤陶色 `#c96442` 作为主要强调色
- **有机圆角设计**：8-32px 的圆角半径营造柔和亲切的感觉
- **环状阴影系统**：使用 `0px 0px 0px 1px` 环状阴影代替传统投影

### 📝 排版系统
- **Serif/Sans 分层**：标题使用衬线体（Georgia），正文使用无衬线体（Arial/system-ui）
- **杂志级行高**：正文采用 1.60 的宽松行高，提供书籍般的阅读体验
- **统一的标题权重**：所有标题使用 500 字重，保持一致的声音
- **响应式字号**：根据屏幕尺寸自动调整标题大小

### 🎯 独特元素
- **引用块设计**：左侧 Terracotta 边框 + 适配背景，突出重要内容
- **代码块样式**：温暖清晰的代码展示，浅色版用象牙白，暗色版用深炭色
- **多级列表编号**：一级阿拉伯数字 (1,2,3) → 二级罗马数字 (i,ii,iii) → 三级字母 (a,b,c)
- **表格悬停效果**：微妙的背景变化提升交互体验

## 📦 安装方法

### 方法一：手动安装
1. 下载 `claude-anthropic.css` 文件
2. 打开 Typora
3. 进入 **偏好设置** → **外观** → **打开主题文件夹**
4. 将 CSS 文件复制到主题文件夹
5. 重启 Typora
6. 在 **偏好设置** → **外观** → **主题** 中选择 "Claude Anthropic"

### 方法二：通过主题管理器
（如果 Typora 支持在线主题市场）
1. 搜索 "Claude Anthropic"
2. 点击安装
3. 启用主题

## 🎨 设计原则

### 色彩系统
```
主背景:    #f5f4ed (Parchment - 羊皮纸)
卡片背景:  #faf9f5 (Ivory - 象牙白)
主要文字:  #141413 (Near Black - 近黑)
次要文字:  #5e5d59 (Olive Gray - 橄榄灰)
强调色:    #c96442 (Terracotta - 赤陶)
边框:      #f0eee6 (Border Cream - 奶油边)
```

### 字体层次
- **H1**: 64px, Serif, 行高 1.10
- **H2**: 52px, Serif, 行高 1.20
- **H3**: 36px, Serif, 行高 1.30
- **H4**: 32px, Serif, 行高 1.10
- **H5**: 25px, Serif, 行高 1.20
- **H6**: 20px, Serif, 行高 1.20
- **正文**: 16px, Sans, 行高 1.60
- **代码**: 15px, Mono, 行高 1.60

### 间距系统
基于 8px 基础单位：3px, 4px, 6px, 8px, 10px, 12px, 16px, 20px, 24px, 30px

```

卡片背景:  #30302e (Dark Surface - 深炭色)
```

两个版本共享：
- ✅ 相同的排版层次和字体系统
- ✅ 相同的间距和圆角规范
- ✅ 相同的组件行为（列表、表格、引用等）
- ✅ 相同的响应式断点
- ✅ 相同的品牌强调色 (#c96442)

## 📱 响应式设计

主题针对不同屏幕尺寸进行了优化：
- **桌面端** (>992px): 完整布局，最大宽度 1200px
- **平板端** (768-991px): 适度缩小标题，保持可读性
- **移动端** (<768px): 进一步压缩间距和字号，确保小屏体验

## 🔧 自定义

如果您想微调主题，可以编辑 CSS 文件中的变量部分：

```css
:root {
  /* 修改主背景色 */
  --parchment: #f5f4ed;  /* 浅色版 */
  /* 或 */
  --parchment: #141413;  /* 暗色版 */
  
  /* 修改强调色 */
  --terracotta-brand: #c96442;
  
  /* 修改正文字号 */
  font-size: 16px;
}
```
