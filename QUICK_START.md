# 🚀 Quick Start - Claude Anthropic Typora Theme

## 3 步开始使用

### Step 1️⃣: 复制主题文件

将 `claude-anthropic.css` 复制到 Typora 主题文件夹

**快捷方式：**
- **macOS**: `Cmd + ,` → Appearance → Open Theme Folder
- **Windows/Linux**: `Ctrl + ,` → Appearance → Open Theme Folder

### Step 2️⃣: 重启 Typora

完全关闭并重新打开 Typora

### Step 3️⃣: 启用主题

Preferences → Appearance → Theme → 选择 **"Claude Anthropic"**

---

## ✅ 验证安装

如果看到以上效果，恭喜！安装成功！🎉

## 🔧 常见问题

**Q: 主题没有出现在列表中？**  
A: 确认文件名是 `claude-anthropic.css`，重启 Typora

**Q: 字体看起来不一样？**  
A: 这是正常的，主题使用系统回退字体（Georgia, Arial）

**Q: 如何自定义颜色？**  
A: 编辑 CSS 文件的 `:root` 部分，修改变量值

**Q: 支持暗色模式吗？**  
A: 是的！当系统设置为暗色模式时自动切换
