## 🎯 设计原则遵循

### ✅ 完全符合规范
1. **温暖优先** ✓
   - 所有灰色都有黄棕底色
   - 无冷蓝灰色

2. **Serif 权威** ✓
   - 所有标题使用衬线体
   - 统一 500 字重

3. **宽松节奏** ✓
   - 正文 1.60 行高
   - Editorial spacing

4. **克制优雅** ✓
   - 无渐变
   - 无重投影
   - 无饱和色（除 Terracotta）

5. **有机圆角** ✓
   - 8-32px 范围
   - 亲切柔和

6. **环状深度** ✓
   - 0px 0px 0px 1px 系统
   - Border-like shadows

## 🚀 使用方法

### 快速安装（3 步）
```bash
1. 复制 claude-anthropic.css 到 Typora 主题文件夹
2. 重启 Typora
3. 选择 "Claude Anthropic" 主题
```

### 验证安装
```bash
在 Typora 中打开 preview.md
检查温暖的羊皮纸背景和 Serif 标题
```

### 开始写作
```bash
享受温暖、优雅的写作体验！
```
